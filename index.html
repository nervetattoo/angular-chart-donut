Angular donut chart

A simple <donut-chart> directive for Angular built with D3.js

Installation

Use bower to download it bower install angular-chart-donut then include the script and the module as a dependency angular.module('mymodule', ['chart.donut']).

Demo


              <chart-donut title="Demo" data="{{data}}" size="250">
